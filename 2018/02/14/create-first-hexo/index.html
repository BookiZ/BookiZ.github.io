create first-hexo

当我看到老师说windows安装hexo很容易安装不成功，我的内心是抓狂的，可是我还是想尝试一下，终于历经很多磨难，安装成功。下面来说一下我的安装步骤。

1. 先说一下安装前的环境配置

  • 安装node.js
  • 安装git
  • 安装markdown(或者安装一款带有markdown插件的IDE,如webstorm)

以上都可在官网下载
2. 详细安装步骤

  • 新建一个文件夹（myblog)
  • 右键该文件，选择Git Bash here （提前安装好git bash工具）
  • 运行命令
  • 安装成功后，运行命令
  • 安装依赖包（我的系统在上一部就自动安装了，我没有运行这一部，看自己的情况哦）
  • 继续运行命令
  • 运行命令


然后你就可以通过，http://localhost:4000/预览啦

不过，这个只能本地查看，接下来我们把它部署到github上

  1. start _config.yml，编辑网站配置
    1. 把第 6 行的 title 改成你想要的名字
    2. 把第 9 行的 author 改成你的大名
    3. 把最后一行的 type 改成 type: git
    4. 在最后一行后面新增一行，左边与 type 平齐，加上一行 repo: 仓库地址 （请将仓库地址改为「你的用户名.github.io」对应的仓库地址，仓库地址以 git@github.com: 开头）

第 4 步的 repo: 后面有个空格

  1. 运行命令
    npm install hexo-deployer-git --save
  2. 运行
    hexo deploy
  3. 进入「你的用户名.github.io」对应的 repo，打开 GitHub Pages 功能，如果已经打开了，就直接点击预览链接
  4. 你现在应该看到了你的博客！

以后我们在hexo上写博客就很简单了。

  1. hexo new newBlogName
  2. 复制显示的路径，使用 start 路径 来编辑它
  3. hexo generate
  4. hexo deploy
  5. 去看你的博客，应该能看到新的博客了
